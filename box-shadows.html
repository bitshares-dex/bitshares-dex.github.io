Box-shadows.css a bunch of simple and pleasant shadows for you to use in your projects. Great for tables, homepages content, containers, navigation and other blocks your website.

Box-shadows.css - это коллекция простых и приятных css теней для ваших проектов. Они отлично подходят для использования в таблицах, для содержимого домашних страниц, контейнеров, навигации и других блоков вашего сайта. Пробуйте, экспериментируйте, используйте как хотите и где хотите!

Now the collection consists of 45 shadows with number, bShadow, bShadow-light and bShadow-inset. Look them up.

Сейчас коллекция насчитывает более 45 вариантов теней с номером, а так же три дополнительных класса .bShadow, .bShadow-light и .bShadow-inset. Нажмите на селектор ниже и посмотрите их в действии. Вы можете переключать тени используя стрелки на клавиатуре. Результат отображается в виде тени в основном контейнере сайта.

1. Change the background of the site and container to your own.

1. Воспользуйтесь формой ниже и измените фон сайта и контейнера на свои собственные.

2. Наводите на блоки, чтобы увидеть их в действии. Результат отображается в блоках и в виде тени основного контейнера сайта.

4. Click "Download" and install the downloaded file according to the "Basic Usage".

4. Нажмите кнопку «Download» и подключите скаченный файл к своему сайту в соответствии с инструкцией.

Basic Usage

1. Include the stylesheet on your document's <head>:

Использование

1. Подключите таблицу стилей на своем сайте, после открывающего тега <head>:

<head>
<link rel="stylesheet" href="/box-shadows.css">
<!-- or -->
<link rel="stylesheet" href="/box-shadows.min.css">
</head>

2. Add the class .bShadow to the block you want to use:

2. Добавьте класс .bShadow к блоку, который вы хотите использовать:

<div class="bShadow"></div>

3. Next, you need to add an additional class to the existing .bShadow with the number. Example, .bShadow-number:

3. Далее вам необходимо добавить номер к уже имеющемуся .bShadow еще класс с номером .bShadow-number:

<div class="bShadow bShadow-1"></div>

or use one of the classes separately, without class .bShadow:

или используйте один из классов отдельно, без .bShadow:

<div class="bShadow-1"></div>
